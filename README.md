## Description
Nabeh “نبيه “is a game that helps users to improve their Arabic in a fun exciting way. The game is level-based, it has three categories to choose from word of the day, the synonyms, and the opposite of words and grammar where the user has the choice of which category to play

## Features
* Multiple levels and challenges
* User-friendly interface
* Progress tracking and scoring system
* Subject-specific categories
* Leaderboards to encourage competition
* Customizable settings and options
